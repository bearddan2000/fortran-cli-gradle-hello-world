# fortran-cli-gradle-hello-world

## Description
This is a POC project to demonstrate a
gradle as a build tool for fortran.

## Tech stack
- gfortran
- gradle
- fortran

## Docker stack
- gradle:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
